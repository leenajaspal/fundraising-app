# Charity Fundraising Application

## How to Run
1. Open a terminal in this folder.
2. Install dependencies:
   npm install
3. Start the server:
   node server.js
4. Open the app in a browser:
   http://localhost:3000

## Notes
- No real payment is processed; the PaymentService is simulated.
- Emails are simulated using console logs.
- Guest users can browse fundraisers but must log in to donate or report.
- The app contains seeded example fundraisers and donations.
